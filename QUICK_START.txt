## 🎪 8 Available Animation Functions

Call these from JavaScript console or your code:

### 1️⃣ Show Toast Notification
```javascript
showToast('Your message here', 'success', 4000);
```
**Types**: `success`, `error`, `warning`, `info`

### 2️⃣ Show Modal Dialog
```javascript
showModal('Title', 'Content message', [
    { text: 'OK', class: 'btn-primary' }
]);
```

### 3️⃣ Show Confirmation
```javascript
showConfirm('Sure?', 'Confirm?',
    () => alert('Confirmed!'),
    () => alert('Cancelled!')
);
```

### 4️⃣ Bounce Animation
```javascript
animateElementBounce(document.querySelector('.element'));
```

### 5️⃣ Shake Animation
```javascript
shakeElement(document.querySelector('.element'));
```

### 6️⃣ Pulse Glow
```javascript
pulseElement(document.querySelector('.element'));
```

### 7️⃣ Rotate In
```javascript
rotateInElement(document.querySelector('.element'));
```

### 8️⃣ Other Animations
```javascript
flipElement(element);              // 3D flip
heartbeatElement(element);         // Heartbeat
makeElementFloat(element);         // Bobbing motion
wiggleElement(element);            // Wiggle effect
```

## ❓ Troubleshooting

### No animations appearing?
- [ ] Check both servers are running (ports 5000 & 8000)
- [ ] Clear browser cache (Ctrl+Shift+Delete)
- [ ] Try animations-demo.html first
- [ ] Check browser console for errors (F12)

### Animations too fast/slow?
- Edit animation duration in `styles.css`
- Change values like `animation: name 0.5s ease-out`
- Duration is in seconds (0.5s = 500ms)

### Want different colors?
- Edit CSS variables in `styles.css`:
  ```css
  :root {
      --success: #22C55E;
      --error: #EF4444;
      ...
  }
  ```

### Toast not showing?
- Check JavaScript console (F12)
- Make sure you're using correct syntax:
  ```javascript
  showToast('Message', 'success', 4000);
  ```
