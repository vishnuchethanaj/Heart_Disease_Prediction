## ⚙️ JavaScript Additions (in `app.js`)

### Toast Notification Function
```javascript
showToast(message, type = 'info', duration = 4000)
```
**Creates animated toast notification**
- Parameters:
  - `message` - Text to display
  - `type` - 'success' | 'error' | 'warning' | 'info'
  - `duration` - Auto-dismiss time in ms (0 = no auto-dismiss)
- Features:
  - Pop-in animation from right
  - Auto-dismisses with fade-out
  - Manual close button
  - Color-coded by type

**Usage:**
```javascript
showToast('Success!', 'success', 4000);
showToast('Error occurred!', 'error', 5000);
```

---

### Modal Popup Function
```javascript
showModal(title, content, buttons = [])
```
**Creates animated modal dialog**
- Parameters:
  - `title` - Modal heading
  - `content` - HTML content (can include HTML)
  - `buttons` - Array of button objects
- Button Format:
  ```javascript
  { text: 'Button Label', class: 'btn-primary', onclick: 'functionName()' }
  ```
- Features:
  - Bounce scale-up animation
  - Overlay blur effect
  - Click outside to close
  - Custom buttons with callbacks

**Usage:**
```javascript
showModal('Confirm', 'Proceed with prediction?', [
    { text: 'Yes', class: 'btn-primary', onclick: 'startPrediction()' },
    { text: 'No', class: 'btn-secondary', onclick: 'cancelPrediction()' }
]);
```

---

### Confirmation Dialog Function
```javascript
showConfirm(title, message, onConfirm, onCancel)
```
**Creates confirm/cancel popup**
- Parameters:
  - `title` - Dialog title
  - `message` - Question/message text
  - `onConfirm` - Callback when confirmed
  - `onCancel` - Callback when cancelled
- Features:
  - Built-in Confirm and Cancel buttons
  - Professional styling
  - Callback-based handling

**Usage:**
```javascript
showConfirm('Delete Data?', 'This cannot be undone.',
    () => { deleteData(); },
    () => { console.log('Cancelled'); }
);
```

### Element Animation Functions

#### Bounce Animation
```javascript
animateElementBounce(element)
```
- Effect: Elastic spring-like entrance
- Duration: 600ms
- Best for: Highlighting successful actions

#### Shake Animation
```javascript
shakeElement(element)
```
- Effect: Horizontal error shake
- Duration: 500ms
- Best for: Validation errors, alerts

#### Pulse Glow Animation
```javascript

```
- Effect: Expanding radial glow
- Duration: 2s (repeats)
- Best for: Important metrics, highlights

#### Rotate In Animation
```javascript
rotateInElement(element)
```
- Effect: 180° spin with fade-in
- Duration: 600ms
- Best for: Transformations, reveals

#### Flip Animation
```javascript
flipElement(element)
```
- Effect: 3D Y-axis flip
- Duration: 600ms
- Best for: Card reveals, state changes

#### Heartbeat Animation
```javascript

```
- Effect: Double-beat scaling
- Duration: 1.3s
- Best for: Vital indicators, heart metrics

#### Floating Animation

- Duration: 3s (repeats indefinitely)
- Best for: Decorative elements, background items

#### Wiggle Animation
```javascript

```
- Effect: Playful side-to-side shaking
- Duration: 500ms
- Best for: Subtle attention-getting

## 🔄 Integration with Existing Code

### Automatic Animations in Prediction Flow

The JavaScript has been enhanced to automatically show animations:

```javascript
// When prediction succeeds:
// 1. Success toast appears
showToast('✓ Prediction completed successfully!', 'success', 5000);

// 2. Result cards slide up with delays
document.querySelectorAll('.result-card').forEach((card, idx) => {
    card.style.animation = `slideUpCard 0.6s ease-out forwards`;
    card.style.animationDelay = `${idx * 0.1}s`;
});

// When prediction fails:
// 1. Error toast appears
showToast('Error: Could not connect to server', 'error', 5000);

// 2. Processing page shakes
shakeElement(document.getElementById('processing-page'));
```

## 📊 Animation Specifications

### Timing
- **Toast**: 0.5s pop-in, 4s display, 0.5s exit
- **Modal**: 0.5s bounce entrance
- **Element animations**: 400-1300ms depending on type
- **Infinite loops**: 2-3 second cycles

### Effects Used
- **Transform**: translateX/Y, scale, rotate
- **Opacity**: Fade in/out transitions
- **Blur**: Backdrop-filter for overlays
- **Box-shadow**: Glow effects
- **Gradients**: Color animations

### Browser Support
- CSS Transforms: 90%+ browsers
- CSS Animations: 95%+ browsers
- Backdrop-filter: 85%+ browsers (graceful degradation)
- All modern mobile browsers supported

## 🎨 Customization Examples

### Change Toast Duration
```javascript
showToast('Quick message', 'info', 2000);  // 2 seconds
showToast('Long message', 'info', 7000);   // 7 seconds
showToast('Persistent', 'info', 0);        // Never auto-dismiss
```

### Change Animation Colors (Edit CSS)
```css
.alert-success {
    background: linear-gradient(135deg, rgba(34, 197, 94, 0.15), rgba(34, 197, 94, 0.05));
    border-color: rgba(34, 197, 94, 0.3);
    color: #4ADE80;
}
```

### Change Animation Duration
```css
@keyframes popInAlert {
    /* Change 0.5s to desired duration */
    animation: popInAlert 0.3s cubic-bezier(...) forwards;  /* Faster */
}
```
