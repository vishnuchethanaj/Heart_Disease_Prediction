How This Project Works

CardioAI is an end-to-end heart disease risk prediction web application combining structured clinical input, ANN-based risk inference, explainable feature importance, and a user-friendly visual dashboard.

1. Clinical Data Input

Collects key patient indicators like age, blood pressure, cholesterol, fasting blood sugar, and max heart rate through validated form controls.

2. Feature Processing

Normalizes and validates selected medical features aligned with your curated dataset for consistent prediction quality.

3. ANN Risk Inference

Backend prediction engine computes probability of disease, confidence score, and interpretable feature contribution values.

Technology Stack
  • Frontend: HTML, CSS, JavaScript, Chart.js
  • Backend: Flask REST API
  • Data Source: Heart disease selected feature dataset
  • Model Logic: ANN-inspired risk scoring and explainability output
Output You Get
  • Risk percentage with low/medium/high categorization
  • Probability distribution and confidence score
  • Feature importance visualization
  • Personalized health roadmap and comparison table
Why It Matters
  • Supports early screening and proactive intervention
  • Presents complex model output in clinician-friendly format
  • Improves awareness through interactive simulation and reporting
  • Built for clear, explainable and actionable risk communication
