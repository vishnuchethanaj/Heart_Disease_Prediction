🔧 CUSTOMIZATION
═══════════════════════════════════════════════════════════════════════════════

Change Colors:
  1. Edit CSS variables in styles.css:
     :root {
         --success: #22C55E;
         --error: #EF4444;
         --warning: #F59E0B;
         --info: #3B82F6;
     }

Change Timing:
  1. Edit animation duration in keyframes:
     @keyframes popInAlert {
         animation: popInAlert 0.5s cubic-bezier(...) forwards;  // Change 0.5s
     }

Change Behavior:
  1. Modify function parameters
  2. Add/remove animation classes
  3. Create new @keyframes
  4. Extend existing functions
